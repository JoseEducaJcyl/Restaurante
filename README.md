# Restaurante 

Proyecto sencillo desarrollado en IntelliJ IDEA para gestionar un restaurante usando Java y Spring Boot.

## ¿Qué hace?

- Añadir, editar y eliminar platos del menú
- Realizar pedidos y generar facturas
  
## Cómo probarlo

1. Clona el proyecto
2. Abre el proyecto en IntelliJ
3. Ejecuta la clase `main.java`
